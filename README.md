# NetLogo Recommendation Dynamics

This repository hosts the NetLogo model [`my-rec-alg.nlogox`](./my-rec-alg.nlogox) and a small GitHub Pages site for students.

## Student link

After GitHub Pages is enabled for the repository, students can use:

- Site page: `https://lisacuk.github.io/netlogo-recommendation-dynamics/`
- NetLogo Web launcher: `https://netlogoweb.org/launch#https://lisacuk.github.io/netlogo-recommendation-dynamics/my-rec-alg.nlogox`

## Local contents

- `my-rec-alg.nlogox`: NetLogo model
- `index.html`: public landing page for GitHub Pages
